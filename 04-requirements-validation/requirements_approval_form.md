## Change Control After Approval

**Important:** Once this form is signed:
- Requirements are BASELINED
- Changes require formal Change Request
- Change Control Board reviews and approves changes
- Impact analysis required for all changes
- Traceability updated for all changes

**Change Request Process:**
1. Submit Change Request Form to Project Manager
2. Business Analyst performs impact analysis (scope, cost, schedule)
3. Change Control Board reviews
4. Approver decision (approve, reject, defer)
5. If approved: Update requirements docs, RTM, and project plan
6. Communicate to all stakeholders
